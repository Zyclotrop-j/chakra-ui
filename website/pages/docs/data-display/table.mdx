`Table` component is used to display tabular data. It renders a `<table>` element by
default.

## Import

```js
import {
  Table,
  THead,
  TH,
  TFoot,
  TD,
  TR,
  TBody,
  Caption
} from "@chakra-ui/react"
```

## Default Table

```jsx
<Table>
    <THead>
    <TR>
        <TH scope="col">#</TH>
        <TH scope="col">First</TH>
        <TH scope="col">Last</TH>
        <TH scope="col">Handle</TH>
    </TR>
    </THead>
    <TBody>
    <TR>
        <TH scope="row">1</TH>
        <TD>Mark</TD>
        <TD>Otto</TD>
        <TD>@mdo</TD>
    </TR>
    <TR>
        <TH scope="row">2</TH>
        <TD>Jacob</TD>
        <TD>Thornton</TD>
        <TD>@fat</TD>
    </TR>
    <TR>
        <TH scope="row">3</TH>
        <TD>Larry</TD>
        <TD>The Bird</TD>
        <TD>@twitter</TD>
    </TR>
    </TBody>
</Table>
```

## Varient overview

You can combine striped and border-x variants.

<Table>
    <THead>
        <TR>
            <TH scope="col">Striped?</TH>
            <TH scope="col">Normal borders</TH>
            <TH scope="col">Full borders</TH>
            <TH scope="col">Borderless</TH>
        </TR>
    </THead>
    <TBody>
        <TR>
            <TH scope="row">Normal</TH>
            <TD>normal</TD>
            <TD>bordered</TD>
            <TD>borderless</TD>
        </TR>
        <TR>
            <TH scope="row">Striped</TH>
            <TD>striped</TD>
            <TD>bordered-striped</TD>
            <TD>borderless-striped</TD>
        </TR>
    </TBody>
</Table>

## Themeing

Tables are fully themeable. The table-theme is a multipart-component-theme and consists of its parts `table`, `thead`, `tbody`, `tfoot`, `tr`, `th`, `td` and `caption`.
See the [Multipart themeing documentation](/docs/theming/component-style#styling-multipart-components) for details.
